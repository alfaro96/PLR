Legend for changelogs
=====================

* **Major Feature**: Something big that you could not do before.
* **Feature**: Something that you could not do before.
* **Efficiency**: An existing feature now may not require as much computation or memory.
* **Enhancement**: A miscellaneous minor improvement.
* **Fix**: Something that previously did not work as documentated, should now work.
* **API Change**: You will need to change your code to have the same effect in the future.
